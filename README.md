# TCA-CDA

**TCA-CDA** 是一个用于预测潜在环状RNA-疾病关联的计算框架。该方法通过联合建模异构生物网络中的多视角生物属性和拓扑上下文对齐表示，提高了环状RNA-疾病关联预测的准确性与鲁棒性。

本存储库包含本文研究中使用的实现代码。

---

## 概述

准确识别潜在的环状RNA与疾病之间的关联，对于理解疾病相关分子机制和发现新的生物标志物具有重要意义。  
TCA-CDA主要整合了以下几个部分：

- 多视角生物特征编码
- 拓扑上下文对齐表示学习
- 复合配对特征构建
- 下游关联预测

通过上述模块的协同建模，TCA-CDA能够有效提高circRNA-疾病关联预测性能。

---

## 框架

TCA-CDA的总体框架由以下四个主要部分组成：

1. **多视角生物特征编码**
   - 环状RNA序列语义表示
   - 疾病功能相似性表示

2. **拓扑上下文对齐块（TCA-Block）**
   - 拓扑感知邻域编码
   - 上下文依赖建模

3. **复合配对特征构建**


4. **关联预测**
   - 双隐藏层感知机分类器
